    Como compilar o APK

    1. Pré-requisitos

    • Node.js 18+ nodejs.org

    • Android Studio developer.android.com

    • JDK 11+ adoptium.net

    2. Instalar Cordova

    $ npm install -g cordova

    3. Extrair e navegar

    $ unzip cordova-project.zip

    $ cd cordova-project

    4. Adicionar plataforma Android

    $ cordova platform add android@latest

    5. Instalar plugins

    $ npm install

    6. Compilar APK

    $ cordova build android

    7. APK gerado em:

    platforms/android/app/build/outputs/apk/debug/app-debug.apk

    Para APK assinado (produção):

    $ cordova build android --release

    # Depois assine com jarsigner ou apksigner

    💡 Dica: Para testar rapidamente, você também pode usar cordova run android com um dispositivo USB conectado ou emulador.
